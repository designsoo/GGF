# 🎮 What is GGF?

### Good Game Friends

함께 게임할 사람을 모집하는 커뮤니티 웹서비스 입니다.

- `롤` / `배그` / `오버워치` / `마인크래프트`
- 아래와 같은 유형의 글을 올릴 수 있습니다.
  - 파티원 모집(오프라인/온라인
  - 클랜원 모집
  - 게임 공략
