# 🎮 What is GGF?

### Good Game Friends

게임을 사랑하는 이들을 위한 특별한 공간으로, 다양한 게임을 즐기며 함께하는 멋진 친구들을 만날 수 있는 커뮤니티 웹사이트입니다.

- `롤` / `배그` / `오버워치` / `마인크래프트`
- 아래와 같은 유형의 글을 올릴 수 있습니다.
  - 파티원 모집(오프라인/온라인)
  - 클랜원 모집
  - 게임 공략

# ❓ How to run this project
```
# repository clone
git clone https://github.com/sprint-team3/GGF.git

# npm 설치
npm install

# dev server
npm run dev

# build
npm run build
npm run start
```
